# Gestion des logiciels métiers

## Description
Ce projet est une application web pour la gestion des logiciels métiers. 
Il permet de gérer les utilisateurs, les licences, les notifications, et le journal des actions.

## Technologies utilisées
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** PHP, Laravel
- **Base de données:** MySQL
- **Modélisation:** UML (diagrammes de classes et cas d'utilisation)

## Fonctionnalités
- Gestion des utilisateurs
- Gestion des licences
- Gestion des logiciels
- Journalisation des actions
- Notifications

## Installation
1. Cloner le repository :
   ```bash
   git clone https://github.com/f-ezzahrae1/gestion-logiciels-metiers.git
Installer les dépendances Laravel :
composer install

Configurer la base de données dans .env
Lancer le serveur Laravel :
php artisan serve
